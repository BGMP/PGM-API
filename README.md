===

API used for PGM persistent data-tracking.

## Installation

#### Prerequisites
* Java 8 or above
* [Maven](http://maven.apache.org/) (Dependency Management)

#### Procedure
As in any other Maven project, simply clone the repository, launch your command prompt within it, and run:

  > `mvn clean package`
